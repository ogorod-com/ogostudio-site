About the game

GeoFlag is a geography game built to make the world feel worth knowing. It covers 154 countries across 4 game modes, with a daily challenge that refreshes every day. The demo is on Steam now and the full game launches July 13, 2026.

Key facts

  • 154 countries
  • 4 game modes: Map Game, Memory, Study Mode, Daily Challenge
  • A daily challenge that refreshes every day
  • Built by one person
  • Family-friendly, no ads, no in-app purchases
